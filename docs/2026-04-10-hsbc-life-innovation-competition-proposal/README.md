# HSBC Life Insurance Innovation Competition 2026

This folder is a working proposal pack for the HSBC Life Insurance Innovation Competition 2026.

Recommended track for GOFA:
- Health & Wellness Ecosystem

Why this track:
- It is the closest fit to GOFA's existing strengths in movement assessment, preventive health engagement, healthy ageing, and insurer-facing wellness journeys.
- HSBC explicitly asks how insurers can move beyond claims, build healthtech and fitness partnerships, and use wearable or health-app data responsibly.

What is inside:
- `competition-brief.md`: official competition facts collected from HSBC's public site
- `proposal-direction.md`: recommended strategic angle for GOFA
- `gofa-fit-and-partnership.md`: how `gofa-web-nextjs` and `sc-gofa-beta1` map to HSBC Life's goals and why GOFA should be framed as a partner
- `executive-summary-for-judges.md`: a one-page judge-facing summary
- `pitch-deck-outline.md`: a 7-slide pitch deck built from the partner narrative
- `proposal-draft.md`: a first-pass proposal draft you can adapt into the final submission
- `submission-checklist.md`: a working checklist for deadlines, attachments, and judging alignment

Suggested use order:
1. Read `competition-brief.md`
2. Review `proposal-direction.md`
3. Read `gofa-fit-and-partnership.md`
4. Start with `executive-summary-for-judges.md`
5. Use `pitch-deck-outline.md` to build slides
6. Edit `proposal-draft.md` into your final version
7. Use `submission-checklist.md` before sending
